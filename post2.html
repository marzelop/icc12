Extensão do Chrome para reconhecimento de texto em imagens no navegador

Eu desenvolvi uma extensão do Chrome que permite copiar e colar textos diretamente de uma imagem no navegador. Utilizei a linguagem Python para criar o código de extração de texto e JavaScript para integrar esse código numa extensão de navegador, além da própria API do Google Chrome.

Para utilizar a extensão é só baixar a extensão na Chrome Web Store ou clonar o repositório no GitHub.

Como Usar?

Para extrair o texto de uma imagem é só você clicar com o botão direito do mouse sobre a imagem e dentro das opções disponibilizadas normalmente haverá uma nova, extrair texto. É só clicar com o botão esquerdo do mouse nessa opção e aparecerá uma caixa de texto com todo o texto detectado na imagem, então é só selecionar o texto que você quer e apertar ctrl+C e depois ctrl+V onde você quiser!

Tecnologias

Foi utilizada a linguagem Python para o código de extração de texto por causa da sua grande disponibilidade de bibliotecas e agilidade no desenvolvimento, já que existem funções para a detecção e transcrição de texto prontas dentro das bibliotecas da linguagem.

Já a escolha do JavaScript foi por conta da sua compatibilidade nativa com os navegadores e também a integração com a API disponibilizada pela desenvolvedora do navegador.
